# 起動方法

## 初回セットアップ（1回だけ）

### 1. Pythonのインストール確認
```
python --version
# Python 3.10以上が必要
```

### 2. 依存パッケージのインストール
```
cd painting_ai_system
pip install -r requirements.txt
```

### 3. APIキーの設定
1. `.env.example` を `.env` にコピー
2. `.env` をメモ帳で開く
3. `OPENAI_API_KEY=sk-...` の部分を自分のAPIキーに変更して保存

OpenAI APIキーの取得：https://platform.openai.com/api-keys

### 4. 会社情報の設定（任意）
`.env` ファイルに会社名・住所・電話番号を入力すると、見積書に自動で印刷されます。

---

## 毎回の起動方法

```
cd painting_ai_system
streamlit run app.py
```

ブラウザが自動で開きます。開かない場合は http://localhost:8501 にアクセス。

---

## 使い方

1. **案件情報**（任意）：お客様名・現場住所・担当者名を入力
2. **写真**：現場写真をまとめてアップロード（10〜30枚推奨）
3. **説明**：施工内容をテキストで入力（コピペOK）
4. **「AI解析スタート」**ボタンを押す
5. 追加確認事項があれば回答する
6. 見積書をExcel/PDFでダウンロード

---

## よくある質問

**Q: 写真なしでも使えますか？**
A: テキスト説明だけでも積算できますが、精度は下がります。

**Q: 単価を変えたいです**
A: `data/unit_prices/default_prices.json` をメモ帳で開いて編集できます。

**Q: エラーが出ます**
A: まず `.env` のAPIキーが正しいか確認してください。
